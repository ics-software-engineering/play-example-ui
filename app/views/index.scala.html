Welcome to the play-example-ui project. The goal of this Play application is to illustrate simple user interface elements using Twitter Bootstrap with a responsive layout.

Note that this project does not use a database backend so that the code can be as simple as possible. In the case of forms, associated model objects are created so that you can see how forms are populated from models and vice-versa.

Simple Form

Shows basic form elements and how they can be connected to model instances.

Validation

Adds validation code to the simple form.

Simple Table

Shows how to create and manipulate tables.

Data Binding

Shows how to use the Play "Custom Data Binding" feature.

Tool Tips

Example of a JavaScript tooltip.

Bootswatch

Bootswatch provides simple theme alternatives.

Date Picker

Use a JavaScript datepicker.

Testing

Show how to do testing on all of this.
